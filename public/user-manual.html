주요 작업별 사용법

1. 예약 관리하기

  • 예약 관리 메뉴 클릭
  • 오늘 예약 목록 확인
  • 상태 변경 (확정/취소/완료)
모든 계정 가능

2. 게시글 작성하기

  • 게시판 관리 → "새 게시글"
  • 카테고리 선택 (공지/이벤트/FAQ)
  • 제목·내용 작성 → 발행

3. 팝업 만들기

  • 팝업 관리 → "새 팝업 추가"
  • 팝업 내용 작성
  • 노출 기간 설정 → 저장
슈퍼/일반관리자 가능

4. 이미지 업로드

  • 파일 관리 → "파일 업로드"
  • 이미지 파일 선택
  • 업로드 완료 후 URL 복사
💡 한번 업로드한 이미지는 CMS 전체에서 재사용 가능!

5. 사용자 관리

  • 사용자 관리 → "새 사용자 추가"
  • 권한 선택 (슈퍼/일반/직원)
  • 사용자 정보 입력 → 생성
슈퍼관리자만 가능

6. SEO 관리

  • 시스템 설정 → "SEO" 탭
  • 메타 제목/설명 수정
  • 키워드 및 OG 이미지 설정
슈퍼/일반관리자 가능

주의사항

🔐 보안

• 사용 완료 후 반드시 로그아웃
• 비밀번호 정기적 변경 권장
• 계정 정보 타인과 공유 금지

📱 모바일

• 메뉴: 왼쪽 상단 ≡ 버튼 터치
• 편집: 더블터치로 수정 모드
• 저장: 작업 후 자동 저장

📁 파일

• 이미지: 50MB 이하만 업로드
• 형식: JPG, PNG, GIF, WEBP, PDF 지원
• 백업: Supabase Storage에 자동 저장
• URL 형식: https://...supabase.co/...

문제 해결

문제상황 해결방법
로그인 안됨 아이디/비밀번호 재확인
권한 오류 메시지 관리자에게 권한 요청
파일 업로드 실패 파일 크기/형식 확인
페이지 안열림 브라우저 새로고침 (F5)
모바일 메뉴 안보임 화면 왼쪽 상단 ≡ 클릭
